# php-helper

常用PHP工具函数和类

composer安装：`composer require jjonline/php-helper`

## 一、函数方法(帮助函数类)

命名空间：`jjonline\helper`

1.1、表单效验

`use jjonline\helper\FilterValid;`

1.2、通用函数

`ues jjonline\helper\Tools;`

二、常用Class类(library)

命名空间：`jjonline\library`
